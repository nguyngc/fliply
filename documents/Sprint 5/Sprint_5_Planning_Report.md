# Sprint Planning Report

## Sprint Number & Dates
**Sprint 5**  
**Duration:** 2 weeks (18/03/2026 - 31/03/2026)

## Sprint Goal
The goal of Sprint 5 is to prepare the **Online Flashcard System for Study** for full multilingual support by implementing user interface (UI) localization. This includes externalizing UI text, enabling language selection, supporting multiple languages (including non-Latin), updating backlog items, and ensuring proper documentation and system readiness for future localization scalability.

## Selected Product Backlog Items
1. Identify and document all translatable UI elements
2. Localize GUI to support at least two non-Latin languages
3. Update product backlog with localization-related user stories
4. Update GitHub README with localization instructions
5. Prepare Sprint review report

## Planned Tasks / Breakdown
- Refactor UI to separate static text into resource files
- Create language resource bundles (properties files)
- Implement language switcher in UI
- Translate UI into at least two non-Latin languages
- Test dynamic language switching and UI rendering
- Adjust layouts for multilingual compatibility
- Update backlog (Trello/Jira) with localization stories
- Define acceptance criteria and estimates
- Update GitHub documentation and UML diagrams
